# v3.5.2

## Highlights

- **Dependency Maintenance**: Updated multiple core packages including React, TypeScript, Vite, and ESLint to their latest versions.
- **Security Validation**: Pinned `tar` to `7.5.13` in both npm overrides and Dockerfile to address upstream vulnerabilities.
- **Production Stability**: Resolved React hook linting violations (`react-hooks/set-state-in-effect`) by batching state updates in `queueMicrotask`, preventing React Compiler memoization errors and ensuring stable dashboard rendering.

## Changed

- Bumped various npm dependencies including `react`, `lucide-react`, `vite`, `typescript`, `eslint`, and `wrangler`.
- Pinned `tar` to `7.5.13` in `package.json` overrides and `Dockerfile` to align with the latest version.

## Fixed

- Remediated multiple `react-hooks/set-state-in-effect` violations across dashboard components and hooks by wrapping `setState` calls in `queueMicrotask` to avoid synchronous re-renders and React Compiler memoization errors.
- Removed unnecessary type assertions in `webhookApi.ts` to satisfy `@typescript-eslint/no-unnecessary-type-assertion`.
